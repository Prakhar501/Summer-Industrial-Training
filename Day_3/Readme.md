### 📈 Visualizations Explored

#### 📍 Area Chart
- Used to visualize trends over time while highlighting the magnitude of values.

#### 🥧 Pie Chart
- Used to represent category-wise percentage distribution.

#### 🍩 Donut Chart
- Similar to a pie chart but provides better readability with a central space.

#### 🎯 Gauge Chart
- Used to compare the current value against a target or maximum value.

#### 🪪 Card
- Displays a single important KPI or metric.

#### 🌊 Waterfall Chart
- Shows how positive and negative values contribute to a final result.

#### 🎛️ Slicer
- Interactive filter used to dynamically filter dashboard visuals.
